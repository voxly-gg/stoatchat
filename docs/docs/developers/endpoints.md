# Endpoints

**We are moving stuff around currently following the rebrand, guidance will follow soon!**

You can connect to the API on the following URLs:

| URL                           |  Release   | Description                 |
| ----------------------------- | :--------: | --------------------------- |
| `https://api.voxly.gg`     | Production | Primary API endpoint        |
| `https://app.voxly.gg/api` | Production | API endpoint for old client |
| `https://voxly.gg/api`     |  Staging   | API endpoint for new client |

You can connect to the events server on the following URLs:

| URL                            |  Release   | Description                    |
| ------------------------------ | :--------: | ------------------------------ |
| `wss://ws.voxly.gg`         | Production | Primary events endpoint        |
| `wss://app.voxly.gg/events` | Production | Events endpoint for old client |
| `wss://voxly.gg/events`     |  Staging   | Events endpoint for new client |
